Introduction

EMP3D is a 3D electromagnetic particle-in-cell code for simulating the interaction of intense laser light with plasma. The code is based on the open-source particle-in-cell code Warp developed at Lawrence Berkeley National Laboratory. EMP3D is designed to simulate the interaction of intense laser light with plasma in the context of laser-driven fusion research.
